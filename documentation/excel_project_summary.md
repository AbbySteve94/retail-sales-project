# Retail Sales Excel Dashboard

## Project Overview

This project analyzes retail sales performance using Microsoft Excel.

The dashboard was built using:

- Excel Formulas
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

---

## Business Questions Answered

1. What is the total revenue generated?
2. Which region generates the highest revenue?
3. Which product category performs best?
4. How does revenue trend over time?
5. Which payment methods are used most frequently?

---

## Key Insights

### Revenue by Region

- West Region generated the highest revenue.
- North Region generated the lowest revenue.

### Revenue by Category

- Electronics was the top-performing category.
- Accessories ranked second.

### Monthly Trend

- January produced the highest revenue.
- March produced the lowest revenue.

### Payment Methods

- Credit Card was the most frequently used payment method.
